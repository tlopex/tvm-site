Detailed Description

Base type of all Relax type information.

Type stores possible type information deduced during compile-time. It encapsulates both static type and runtime information such as shape.

Type of each non-primitive Expr can be deduced during compilation in a "best-effort" manner.

When ty appears in function parameter and return signatures, it implies a runtime check that matches the type information with the value.

When it appears in Expr, it follows "assume-semantics", which means the compiler will take the deduced information as it is and only do best effort proofs and checks.

Each type can be uniquely erased to a static-type. The compiler will still compile the code, with less information, when we erase to the static type.

If a Type contains an Expr field, then that field must already be normalized through NormalizeArg. This invariant is checked in constructors and simplifies assumptions during type deduction.

Opaque object.
